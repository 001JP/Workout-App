# Work Out App

Workout app developed in kotlin.


![Logo](https://i.imgur.com/Lk4Ss5M.png)


## Tools and Languages

- Kotlin
- Lottie Animation
- Android Studio
- Git, Github


## Features

- 12 work out animations
- Audio guide
- BMI calculator
- Workout history
